## Installation

```sh
# chsh -s /bin/bash
$ git clone https://github.com/itsjesusurias/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ chmod -R +x **/*.sh
$ chmod -R +x *.sh
$ chmod +x setup.sh
$ chmod +x setup-new-machine.sh
$ ./setup-new-machine.sh

# install vscode extensions
$ ./vscode/extensions.sh

# install npm
$ ./install/npm.sh
```
